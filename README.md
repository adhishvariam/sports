## Sports Venues Data Management

This is a Python application that fetches sports venues from API,
sort them by distance, groups them by sports, and searching
through CLI and simple Flask UI.

## Features

1- Fetch data from API

2- Sort by distance

3- Group by sport

4- Search venues

5- CLI interface

6- Flask UI

## Requirements

1- Python

2- requests

3- flask

4- python-dotenv

## Setup

1- Clone the repository

2- Install dependencies:(pip install -r requirements.txt)

3- Run CLI:(python app.py)

4- Run FlaskUI:(python view.py)
